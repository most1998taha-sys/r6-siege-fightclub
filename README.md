# R6 Siege Court

A sarcastic Rainbow Six Siege squad website for Chico.Panda, Chico.Banza, and LongLiveSaleem.

## Local run
npm install
npm run dev

## Deploy to Vercel
1. Upload this folder to a GitHub repository.
2. Go to Vercel.
3. Click Add New > Project.
4. Import the GitHub repository.
5. Keep defaults:
   - Framework: Vite
   - Build command: npm run build
   - Output directory: dist
6. Click Deploy.
